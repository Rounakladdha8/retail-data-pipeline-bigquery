# Retail Data Pipeline (BigQuery)

## Overview
Built an end-to-end retail analytics pipeline using BigQuery to transform raw e-commerce data into analytics-ready datasets.

## Architecture
Raw → Staging → Final

## Data
- orders
- customers
- order_items
- products
- payments

## Transformations
- Joins across multiple tables
- Aggregations to compute revenue and metrics
- CTEs for structured queries
- Window functions for ranking

## Insights
- Revenue by state
- Average order value
- Ranking of high-value orders

## Tech Stack
- BigQuery
- SQL
